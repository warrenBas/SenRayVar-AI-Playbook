# SenrayVar AI Camera Examples

Welcome to the official examples repository for the **SenrayVar AI Camera**. This project is a curated collection of hands-on tutorials and code samples designed to help you integrate high-performance computer vision into your robotics projects.

Whether you are a LEGO® education enthusiast or a professional maker using Arduino/Python, you will find the resources you need here.

🌐 **More Resources:** You can also find additional tutorials and documentation on our [Official Documentation Portal](https://robot-code.com/docs/en/).

---

## 📂 Repository Structure

The repository is organized by platform to ensure a smooth experience for different user groups:

### 1. [LEGO_Spike_Prime_Examples](./LEGO_Spike_Prime_Examples/)
Dedicated to LEGO® SPIKE Prime and Mindstorms Robot Inventor users.
* **No SDK Required:** Uses the Camera's built-in Apps (Color Tracking, Face Detection, etc.) directly with SPIKE's standard MicroPython.
* **Plug & Play:** Examples focus on logic, navigation, and interaction.

### 2. [Arduino_Examples](./Arduino_Examples/)
For makers and developers using the Arduino IDE.
* **SDK Based:** Utilizes the SenrayVar C++ SDK for low-level control and advanced data parsing.
* **Compatible:** Supports ESP32, Arduino Uno/Mega, and other popular boards.

---

## 🚀 Featured Project: Ball Track (LEGO Spike)

Located in: `LEGO_Spike_Prime_Examples/Offical APP/Ball_Track/`

This project demonstrates a fully autonomous "Search and Clear" robot. It uses the camera to find balls and clean them off the field.

## 📸 About SenrayVar AI Camera
SenrayVar is a versatile AI vision sensor designed for education and prototyping. 

* **Capabilities:** Color/Shape/Tag Recognition, Line Tracking, and Custom AI Model deployment.
* **Interface:** Supports Serial (UART) and I2C communication.
